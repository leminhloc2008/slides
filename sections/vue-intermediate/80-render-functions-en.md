# Render functions

## Render functions

concept from React: _JSX_ - combination of JavaScript and XML

Render functions may be used as an alternative to templates

## Render function

```js
export default {
  name: 'Foo',
  render() {
    return (
      <div>
        foo
        <button>bar</button>
      </div>
    );
  },
};
```
