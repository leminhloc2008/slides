## Aktivierungsfunktionen

- ReLU (Rectified Linear Unit)
- Softmax - oft im letzen Layer für Klassifikation verwendet
- Sigmoid - oft im letzen Layer für "Tagging" verwendet (Tags können sich überlappen)
