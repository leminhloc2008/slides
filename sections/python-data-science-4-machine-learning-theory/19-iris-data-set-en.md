# Iris dataset

## Iris dataset

_Iris dataset_: simple example dataset for machine learning / data science

contains measurements of 150 iris plants: 3 different species with 50 samples each

## Iris dataset

properties in the dataset:

- _sepal length_
- _sepal width_
- _petal length_
- _petal width_
- _species name_
