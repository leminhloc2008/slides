## FakeQL

[https://fakeql.com/](https://fakeql.com/)

Template für einfache Todos bei FakeQL:

```json
{
  "todos": [
    { "id": 1, "title": "Go shopping", "completed": true },
    { "id": 49, "title": "Do taxes", "completed": false },
    { "id": 50, "title": "Do dishes", "completed": false }
  ]
}
```
