# Tupel

## Tupel

Erstellung: Einträge werden mit Kommas getrennt, _üblicherweise_ mit runden Klammern umschlossen.

```py
empty_tuple = ()
single_value = ('Thomas', )
single_value = 'Thomas',
two_values = ('Thomas', 'Smith')
two_values = 'Thomas', 'Smith'
```

## Unpacking von Tupeln

```py
time = (23, 45, 0)

hour, minute, second = time
```

Tauschen von Variablennamen:

```py
a, b = b, a
```
