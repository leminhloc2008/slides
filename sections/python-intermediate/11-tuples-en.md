# Tuples

## Creating tuples

Entries are separated by commas, _usually_ surrounded by round brackets.

```py
empty_tuple = ()
single_value = ('Thomas', )
single_value = 'Thomas',
two_values = ('Thomas', 'Smith')
two_values = 'Thomas', 'Smith'
```

## Unpacking of tuples

```py
time = (23, 45, 0)

hour, minute, second = time
```

swapping variables:

```py
a, b = b, a
```
