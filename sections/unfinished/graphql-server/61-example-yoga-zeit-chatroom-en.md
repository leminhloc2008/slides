# Example: a chatroom API

### build a chatroom API with GraphQL and yoga, hosted on zeit.co

## start the project

create a new folder for the project and initialize an npm project:

```bash
npm init
```

## install & set up micro

```bash
npm install micro
```

edit package.json:

```json
{
  "scripts": {
    "start": "micro"
  }
}
```
