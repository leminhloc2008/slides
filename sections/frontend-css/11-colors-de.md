# Farben

## Farben

```css
h1 {
  color: red; /* Schriftfarbe */
  background-color: blue; /* Hintergrundfarbe */
}
```

## Farbangaben

- Standard-Farben  
  z.B. `grey`, `blue`, `lightblue`, ...
- RGB-Definition (rot-grün-blau)  
  z.B. `rgb(255, 128, 128)`
- HEX-Definition  
  z.B. `#ff8080`
- HSL-Definition  
  (hue, saturation, lightness - Farbton, Sättigung, Helligkeit)  
  z.B. `hsl(180, 60%, 70%)`
