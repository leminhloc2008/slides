# Positioning

## Absolute positioning

Example: a `div` should be positioned `10px` from the bottom right corner of its parent

## Absolute positioning

```html
<div id="outer">
  <div id="inner"></div>
</div>
```

```css
#outer {
  position: relative;
}

#inner {
  position: absolute;
  bottom: 10px;
  right: 10px;
}
```

## Positioning

- `position`: `absolute`
- (`position`: `relative`)
- (`position`: `fixed`)
- (`position`: `static`: default value)
