# Inline elements and block elements

## Inline elements and block elements

inline elements:

- arranged next to each other
- as wide as their content
- e.g. `a`, `em`, `strong`, (`img`)

block elements:

- arranged underneath each other
- as wide as possible
- e.g. `h1`, `ul`, `li`, `p`

## span and div

`span` = generic inline element

`div` = generic block element
