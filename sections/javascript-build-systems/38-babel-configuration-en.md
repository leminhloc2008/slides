# Babel configuration

## Babel configuration

npm packages:

- _@babel/core_
- _@babel/preset-env_ (modern JS)
- _@babel/preset-react_ (JSX)
- _@babel/typescript_

## Babel configuration

possible configuration - e.g. in _package.json_:

```json
  "babel": {
    "presets": [
      "@babel/preset-env",
      "@babel/preset-react"
    ]
  },
```
