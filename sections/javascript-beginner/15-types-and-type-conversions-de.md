# Typen und Typenumwandlungen

## Typen

Bestimmen des Typs einer Variable mittels `typeof`:

```js
let answer = '42';

typeof answer;
```

## Typenumwandlungen

Umwandlung in andere Datentypen: `Number(...)`, `String(...)`, `Boolean(...)`, ...

Alternativen: `.toString()`, `parseInt()`, `parseFloat()`
