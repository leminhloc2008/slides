# Listen

## Listen

- `ul` (unordered list)
- `ol` (ordered list - nummeriert)
- `li` (list item)

## Listen

Beispiel:

```html
<h1>tetrapods:</h1>
<ul>
  <li>amphibians</li>
  <li>reptiles</li>
  <li>birds</li>
  <li>mammals</li>
</ul>
```

## Listen

verschachtelte Liste:

```html
<h1>vertebrate:</h1>
<ul>
  <li>fish</li>
  <li>
    tetrapods
    <ul>
      <li>amphibians</li>
      <li>reptiles</li>
      <li>birds</li>
      <li>mammals</li>
    </ul>
  </li>
</ul>
```
